## jar包缝合教程：暴力缝合，非代码修改。

准备工作：MT管理器或者NP管理器，B包要用的包，A包要导出的包。

开始：A包中merge重命名，展开重命名的merge，选中需要导出的文件，导出。打开B包，导入刚才导出的重命名的merge文件，保存退出。B包就是缝合好的包。
